# 🎓 School Management System

A simple **School Management System** built with **JavaScript (Node.js)** to practice programming fundamentals, problem solving, and clean code principles.

This project is a console-based application that allows users to manage student records through an interactive menu-driven interface.

---

## 🚀 Features

* ➕ Add a new student
* 📋 Display all students
* 🔍 Search students by ID
* 🔍 Search students by Name
* ✏️ Update student information
* 🗑️ Delete students
* 📊 View statistics
* 🔄 Sort students

  * Name (A → Z)
  * Grade (Highest / Lowest)
  * Age (Youngest / Oldest)

---

## 📊 Statistics

The system can calculate:

* Total Students
* Passed Students
* Failed Students
* Highest Grade
* Lowest Grade
* Average Grade
